常用正则

  1. 汉字：^[\u4e00-\u9fa5]{0,}$
  2. Email：^\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*$
  3. URL：^https?://([\w-]+.)+[\w-]+(/[\w-./?%&=]*)?$
  4. 手机号码：^1\d{10}$
  5. 身份证号：^(\d{15}|\d{17}(\d|X))$
  6. 邮政编码：[1-9]\d{5}(?!\d) (邮政编码为6位数字)
  7. 密码验证：(?!^[0-9]+$)(?!^[A-z]+$)(?!^[^A-z0-9]+$)^[^\s\u4e00-\u9fa5]{6,16}$
  8. IP验证： ^(\d{1,2}|1\d\d|2[0-4]\d|25[0-5])(\.(\d{1,2}|1\d\d|2[0-4]\d|25[0-5])){3}$
